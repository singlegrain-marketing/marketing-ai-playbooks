# Skills Directory

Marketing AI skills that auto-load in Claude Code via `/skill-name` commands.

## Available Skills

| Command | Category | Description |
|---------|----------|-------------|
| `/copywriting` | Content | Write marketing copy for any page — homepage, landing, pricing, feature, about |
| `/fb-ads-creative` | Paid Media | Generate 60+ Hormozi-style ad hooks and creatives in one session |
| `/paid-ads` | Paid Media | Full campaign strategy, targeting, and optimization across platforms |
| `/programmatic-seo` | SEO | Build SEO pages at scale using templates and data (location, comparison, directory) |
| `/page-cro` | CRO | Analyze any page and get scored conversion optimization recommendations |
| `/seo-audit` | SEO | Technical + on-page SEO audit with prioritized action plan |
| `/competitor-alternatives` | SEO / Sales | Create vs pages, alternative pages, and competitive comparison content |

## Skill File Format

Each skill follows this structure:

1. **Claude.ai Quick Start** — copy-paste block for non-Claude-Code users
2. **When to Use / Initial Assessment** — triggers and context gathering
3. **Framework** — the complete methodology
4. **Quality Gates** — must-pass checklist before delivery
5. **Scoring** — 0-100 rubric for objective quality assessment
6. **Related Skills** — what to use next or instead

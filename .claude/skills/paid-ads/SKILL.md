## Before Starting

Gather this context (ask if not provided):

### 1. Campaign Goals
- Primary objective? (Awareness, traffic, leads, sales)
- Target CPA or ROAS?
- Monthly/weekly budget?
- Constraints? (Brand guidelines, compliance, geographic)

### 2. Product & Offer
- What are you promoting?
- Landing page URL?
- What makes this offer compelling?
- Any promotions or urgency elements?

### 3. Audience
- Who is the ideal customer?
- What problem does your product solve for them?
- Do you have existing customer data for lookalikes?

### 4. Current State
- Have you run ads before? What worked/didn't?
- Do you have existing pixel/conversion data?
- Current funnel conversion rate?

## Platform Selection Guide

### Google Ads
**Best for:** High-intent search traffic, capturing existing demand
- Search, Performance Max, Display, YouTube, Demand Gen

### Meta (Facebook/Instagram)
**Best for:** Demand generation, visual products, broad targeting
- Advantage+ Shopping, Lead Gen, Conversions, Traffic

### LinkedIn Ads
**Best for:** B2B targeting, reaching decision-makers
- Sponsored Content, Message Ads, Lead Gen Forms, Document Ads

### Twitter/X Ads
**Best for:** Tech audiences, real-time relevance, thought leadership

### TikTok Ads
**Best for:** Younger demographics, viral creative, brand awareness

## Ad Copy Frameworks

### Problem-Agitate-Solve (PAS)
```
[Problem statement]
[Agitate the pain]
[Introduce solution]
[CTA]
```

### Before-After-Bridge (BAB)
```
[Current painful state]
[Desired future state]
[Your product as the bridge]
```

### Social Proof Lead
```
[Impressive stat or testimonial]
[What you do]
[CTA]
```

### Headline Formulas

**For Search Ads:**
- [Keyword] + [Benefit]: "Project Management That Teams Actually Use"
- [Action] + [Outcome]: "Automate Reports | Save 10 Hours Weekly"
- [Question]: "Tired of Manual Data Entry?"

**For Social Ads:**
- Hook with outcome: "How we 3x'd our conversion rate"
- Hook with curiosity: "The reporting hack no one talks about"
- Hook with contrarian: "Why we stopped using [common tool]"

## Campaign Structure

```
Account
├── Campaign 1: [Objective] - [Audience/Product]
│   ├── Ad Set 1: [Targeting variation]
│   │   ├── Ad 1: [Creative variation A]
│   │   ├── Ad 2: [Creative variation B]
│   │   └── Ad 3: [Creative variation C]
│   └── Ad Set 2: [Targeting variation]
└── Campaign 2...
```

### Naming Conventions
```
[Platform]_[Objective]_[Audience]_[Offer]_[Date]
Example: META_Conv_Lookalike-Customers_FreeTrial_2026Q1
```

### Budget Allocation
- **Testing phase (weeks 1-4):** 70% proven, 30% testing
- **Scaling phase:** Consolidate into winners, increase 20-30% at a time

### Google: Keywords + audience layering + RLSA
### Meta: Core interests → Custom audiences → Lookalikes (start 1%, expand to 3%)
### LinkedIn: Job function + seniority + company size

## Optimization Playbook

**If CPA is too high:**
1. Check landing page conversion rate
2. Tighten audience targeting
3. Test new creative angles
4. Improve ad relevance/quality score
5. Adjust bid strategy

**If CTR is low:**
- Test new hooks/angles
- Refine targeting
- Refresh creative (ad fatigue)
- Improve value proposition

**If CPM is high:**
- Expand targeting
- Try different placements
- Improve creative relevance
- Adjust bid caps

## Retargeting Strategy

| Stage | Audience | Window | Message |
|-------|----------|--------|---------|
| Hot | Cart/trial abandoners | 1-7 days | Urgency, objection handling |
| Warm | Key page visitors | 7-30 days | Case studies, demos |
| Cold | Any site visitor | 30-90 days | Educational content |

Always exclude: existing customers, recent converters, bounced visitors (<10 sec).

## Quality Gates

Before launching any campaign:

- [ ] Conversion tracking installed and verified
- [ ] UTM parameters on all URLs
- [ ] Landing page loads in under 3 seconds
- [ ] Ad copy fits character limits per platform
- [ ] Audiences don't overlap (competing with yourself)
- [ ] Negative keyword lists built (Search campaigns)
- [ ] Budget set with daily caps
- [ ] Exclusions set (existing customers, irrelevant audiences)

---

## Weekly Review Checklist

- [ ] Spend vs. budget pacing
- [ ] CPA/ROAS vs. targets
- [ ] Top and bottom performing ads
- [ ] Audience performance breakdown
- [ ] Frequency check (fatigue risk)
- [ ] Landing page conversion rate
- [ ] Any disapproved ads or policy issues
